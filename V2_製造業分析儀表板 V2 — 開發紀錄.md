### 新增功能

| 功能 | 說明 |
|------|------|
| 全域日期篩選 | 頂部兩個 date input，所有 render 函式依日期範圍過濾資料 |
| 明暗模式切換 | CSS 變數驅動，localStorage 記憶偏好，一鍵切換 |
| 懶載入渲染 | 只渲染當前頁面（V1 每次渲染全部 10 頁） |
| 鑽取彈窗 | Chart.js onClick → 彈出 modal 顯示明細表格 |
| 瀑布圖 | Chart.js 浮動長條技法，用於成本差異分解 |
| 匯出功能 | JSON / Excel（SheetJS）/ PDF（html2canvas + jsPDF）/ PNG |
| 資料上傳驗證 | 上傳 Excel 後顯示驗證報告 modal（各 sheet 筆數、欄位檢查） |
| 異常偵測引擎 | 庫存/採購/成本三域自動評分，紅黃綠燈號 |
| 線性回歸預測 | 最小二乘法，預測未來 3 個月趨勢 |
| DAX 知識庫 | 17 個公式，分 5 類，可搜尋、分類瀏覽 |
| 側邊欄收合 | 響應式設計，小螢幕自動收合 |

### V1 問題修正

1. **寫死數據**：V1 多個 render 函式使用硬編碼陣列（如 `[120, 125, 122, 130, 128]`），V2 全部改為從 `currentData` 即時計算
2. **全頁渲染**：V1 每次 `renderDashboard()` 渲染全部 10 頁，V2 改為懶載入（只渲染當前頁）
3. **無響應式**：V2 新增側邊欄收合按鈕與 CSS 響應式斷點

## 技術架構

### 圖表類型對照

| 圖表類型 | 使用頁面 | 實作方式 |
|---------|---------|---------|
| 折線圖 | 庫存趨勢、良率趨勢、準時率趨勢、預測 | `initLineChart()` |
| 長條圖 | 供應商排行、庫齡分佈、差異排行 | `initBarChart()` |
| 環形圖 | ABC 分類、物料類型、成本結構 | `initDoughnutChart()` |
| 雷達圖 | 供應商評分、異常預警 | `initRadarChart()` |
| 散佈圖 | 訂購量 vs 遲交天數 | `initScatterChart()` |
| 堆疊面積圖 | 料工費趨勢 | `initStackedAreaChart()` |
| 瀑布圖 | 成本差異分解 | `initWaterfallChart()`（浮動長條） |
| 熱力圖 | 良率（產品×產線） | HTML table + CSS 背景色 |

### 資料流

```
generateSampleDataV2() → currentData（7 集合）
                              ↓
                    filterByDate() 日期篩選
                              ↓
                    renderMap[activeTab]() 懶載入渲染
                              ↓
                    Chart.js 圖表 + HTML 表格
```
