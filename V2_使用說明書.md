# 製造業分析儀表板 V2 — 使用說明書

## 快速啟動

1. 雙擊 `Manufacturing_Dashboard_V2.html` 在瀏覽器中開啟（推薦 Chrome / Edge）
2. 頁面自動載入內建模擬資料（12 供應商、15 物料、300 採購單、150 生產單）
3. 點擊左側導覽列切換 14 個分析頁面

無需安裝任何伺服器、套件或依賴環境。

---

## 介面導覽

### 左側導覽列（Sidebar）

分為 5 大群組，共 14 個頁面：

| 群組 | 頁面 |
|------|------|
| 總覽 | 首頁總覽、異常預警雷達 |
| 庫存分析 | 庫存效能、呆滯與風險 |
| 供應商分析 | 價格與降本、採購時效與交期、供應商評分卡 |
| 成本分析 | BOM 結構解析、生產數據與良率、成本組成結構、成本差異追蹤、根因分析鑽取 |
| 進階工具 | 滾動預測、DAX 知識庫 |

小螢幕下側邊欄會自動收合，點擊漢堡選單圖示可展開。

### 頂部控制列（Topbar）

| 元素 | 功能 |
|------|------|
| 日期範圍 | 兩個日期輸入框，篩選所有頁面的資料範圍 |
| 重置 | 清除日期篩選，恢復全部資料 |
| 上傳 | 上傳 Excel/CSV 檔案匯入自訂資料 |
| 匯出 ▾ | 下拉選單：JSON / Excel / PDF / PNG |
| 🌙/☀️ | 明暗模式切換（偏好自動記憶） |

## 各頁面功能說明

### 1. 首頁總覽
- 4 個 KPI 卡片：總庫存金額、供應商數、平均良率、平均遲交天數
- 成本趨勢折線圖（近 12 月）
- ABC 分類環形圖
- Top 5 供應商長條圖
- 異常摘要雷達圖

### 2. 異常預警雷達
- 三域燈號：庫存健康度、供應商風險、成本異常（紅/黃/綠）
- 雷達圖：六維度異常評分
- 月度異常趨勢長條圖
- Top 10 異常訂單明細表

### 3. 庫存效能
- KPI：期末庫存、在檔 SKU、動銷率
- 月度庫存趨勢折線圖（從快照資料計算）
- ABC 分類環形圖
- Top 10 高價值庫存表（可點擊鑽取）

### 4. 呆滯與風險
- KPI：超期庫存金額、超期佔比、專用料數量
- 庫齡分佈長條圖（0-30天 / 31-60天 / 61-90天 / >90天）
- 物料類型環形圖（通用料/專用料/非生產用料）
- 消耗率排行長條圖

### 5. 價格與降本
- KPI：採購總金額、供應商數、平均單價
- 帕雷托雙軸圖（金額長條 + 累計佔比折線）
- 採購類別環形圖

### 6. 採購時效與交期
- KPI：平均採購周期、遲交率、平均遲交天數
- 散佈圖（訂購量 vs 遲交天數，識別問題供應商）
- 遲交天數 Top 供應商長條圖
- 月度準時率趨勢折線圖

### 7. 供應商評分卡
- KPI：Preferred / Approved / At-Risk 供應商數量
- 評分卡表格（品質/服務/技術/準時率/Tier）
- 點擊供應商名稱 → 雷達圖即時更新顯示該供應商四維評分

### 8. BOM 結構解析
- KPI：產品數、BOM 項目數、平均 BOM 深度
- 堆疊長條圖（每產品的料/工/費結構）
- BOM 矩陣表（產品 × 原料 × 用量）

### 9. 生產數據與良率
- KPI：總產量、平均良率、不良品數
- 良率趨勢折線圖（含 97% 目標線）
- 產線平均良率長條圖
- 良率熱力圖（產品 × 產線，顏色深淺表示良率高低）

### 10. 成本組成結構
- KPI：總成本、材料佔比、人工佔比
- 堆疊面積趨勢圖（料/工/費 12 個月變化）
- 成本結構環形圖

### 11. 成本差異追蹤
- KPI：材料差異、人工差異、製造費用差異
- 瀑布圖（標準成本 → 材料差異 → 人工差異 → 製造費用差異 → 實際成本）
- 異常訂單表（含旗幟圖示：🏳️ < 1%、🟡 1-2%、🟠 2-5%、🔴 > 5%）

### 12. 根因分析鑽取
- 維度選擇器：按供應商 / 按產線 / 按產品
- 差異排行長條圖（依選擇維度分組）
- 瀑布分解圖（選定維度的料/工/費差異）

### 13. 滾動預測
- 統計摘要：斜率、截距、R² 決定係數
- 預測折線圖：實際值（藍）+ 趨勢線（灰虛線）+ 3 個月預測（橘虛線）
- 使用最小二乘法線性回歸

### 14. DAX 知識庫
- 17 個 Power BI DAX 公式
- 5 個分類：日期表、庫存類、供應商類、生產成本類、成本差異類
- 即時搜尋（輸入關鍵字即時過濾）
- 分類頁籤切換

## 自訂資料上傳

點擊頂部「上傳」按鈕，選擇 Excel 檔案（.xlsx / .xls）。

### 預期工作表格式

| 工作表名稱 | 必要欄位 | 說明 |
|-----------|---------|------|
| Materials | id, name, unitCost, stockQty, abcClass, daysInStock, type, monthlyUsage | 原料主檔 |
| Suppliers | name, category, rating, onTimeDeliveryRate, tier, qualityScore, serviceScore, techScore | 供應商主檔 |
| Products | productId, productName, bomMaterialId, bomQtyPerUnit | 成品 + BOM |
| PurchaseOrders | poId, supplierName, materialId, qty, pricePerUnit, totalAmount, orderDate, plannedDeliveryDate, actualDeliveryDate, delayDays | 採購訂單 |
| ProductionOrders | orderId, productId, productName, line, inputQty, goodOutput, yieldRate, stdMaterialCost, actMaterialCost, materialVar, laborVar, overheadVar, actTotalCost | 生產工單 |
| InventorySnapshots | month, materialId, closingQty, closingValue | 月度庫存快照 |
| CostTypes | level1, level2, level3 | 成本類型層級 |

上傳後會顯示驗證報告，列出各工作表的筆數與狀態。未提供的工作表將保留原始模擬資料。

可先點擊「匯出 → Excel」下載目前資料結構作為範本。

## 匯出功能

| 格式 | 說明 |

## 明暗模式

點擊頂部右側的 🌙/☀️ 按鈕切換。偏好設定自動儲存至瀏覽器 localStorage，下次開啟時自動套用。

## 常見問題

### 圖表無法顯示
- 確認網路連線正常（Chart.js 等套件從 CDN 載入）
- 開啟瀏覽器開發者工具（F12）查看 Console 錯誤訊息

### 上傳 Excel 後資料未更新
- 確認工作表名稱完全一致（區分大小寫）
- 確認欄位名稱與上方表格一致
- 查看驗證報告 modal 中的錯誤提示

### PDF/PNG 匯出空白
- 等待頁面圖表完全渲染後再匯出
- 部分瀏覽器可能阻擋 html2canvas，建議使用 Chrome
