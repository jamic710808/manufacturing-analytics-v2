## 二、V4 願景：智慧驅動的主動式分析平台

```
核心理念：從被動式報表 → AI 驅動的主動式分析

四大轉型：
1. AI 異常偵測 — Isolation Forest 即時發現異常，無需人工設定閾值
2. 需求預測 — Prophet + ARIMA 預測未來庫存與需求
3. 行動建議 — 異常直達解決方案，而非僅展示問題
4. 即時串流 — WebSocket 對接工廠資料，告別手動刷新
```

## 六、AI 功能模組

### 6.1 異常偵測引擎

```
異常偵測類型：
┌────────────────┬────────────────────────────┐
│ 異常類型        │ 偵測演算法                │
├────────────────┼────────────────────────────┤
│ 庫存異常       │ Isolation Forest          │
│ 價格異常       │ Z-Score + LSTM           │
│ 交期異常       │ KNN + DBSCAN             │
│ 良率異常       │ CUSUM + EWMA            │
│ 成本差異異常    │ VAR (向量自我回歸)       │
└────────────────┴────────────────────────────┘
```

### 6.2 需求預測引擎

```python
# Prophet + ARIMA 混合預測
def predict_demand(material_id, horizon_days=90):
    # 短期 → Prophet, 長期 → ARIMA
    prophet = self.prophet.predict(horizon_days)
    arima = self.arima.predict(horizon_days)
    return 0.6 * prophet + 0.3 * arima + 0.1 * external_factors

# 安全庫存計算
def safety_stock(demand, lead_time, service_level=0.95):
    z = norm.ppf(service_level)
    return z * np.sqrt(demand.std**2 * lead_time + lead_time**2 * demand.mean**2)
```
