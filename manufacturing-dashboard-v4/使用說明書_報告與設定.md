## 1. 報告與設定入口

### 1.1 如何進入

在左側導航選單中，找到**「報告與設定」**群組，點擊即可展開子選單：

```
報告與設定
├── 📋 報告生成
└── ⚙️ 系統設定
```

### 1.2 兩種子頁面

| 頁面 | 功能 |
|------|------|
| 📋 報告生成 | 一鍵生成多元分析報告，支援 PDF/Excel/PPTX 格式 |
| ⚙️ 系統設定 | 自訂系統行為與顯示偏好 |

## 2. 報告生成器

### 2.1 頁面位置

**URL**：`/reports/generator`

### 2.2 功能總覽

報告生成器讓您能夠快速生成以下六種專業分析報告：

| 報告類型 | 圖示 | 說明 |
|---------|------|------|
| 經營總覽報告 | 📊 | 包含所有核心 KPI 與趨勢分析 |
| 庫存分析報告 | 📦 | 庫存水位、週轉率、風險評估 |
| 生產效能報告 | 🏭 | OEE、良率、產能分析 |
| 成本分析報告 | 💰 | 成本結構、差異分析、優化建議 |
| 供應商評估報告 | 🚚 | 供應商評分、交期、風險 |
| AI 預測報告 | 🤖 | 需求預測、異常偵測、優化建議 |

### 2.3 操作步驟

#### Step 1：選擇報告類型

在「選擇報告類型」區塊中，點擊您要生成的報告類型。

- **單選**：點擊一次即選擇，再點一次取消選擇
- **多選**：可同時選擇多個報告類型一起生成
- 已選擇的報告會顯示**「✓ 已選擇」**標示

```
📊 經營總覽報告     ← 點擊選擇
📦 庫存分析報告     ← 已選擇（藍色邊框）
🏭 生產效能報告     ← 點擊選擇
💰 成本分析報告
🚚 供應商評估報告
🤖 AI 預測報告
```

#### Step 2：設定報告期間

在「報告設定」區塊中，設定報告的日期範圍：

1. 點擊**開始日期**輸入框，選擇起始日期
2. 點擊**結束日期**輸入框，選擇結束日期

> **提示**：日期範圍會影響報告中的數據統計期間

#### Step 3：選擇輸出格式

支援三種輸出格式，點擊選擇其一：

| 格式 | 說明 | 適用場景 |
|------|------|----------|
| **PDF** | PDF 文件格式 | 正式報告、會議簡報 |
| **Excel** | Excel 試算表格式 | 資料分析、進一步處理 |
| **PPTX** | PowerPoint 簡報格式 | 客戶匯報、簡報演示 |

#### Step 4：（選用）快速範本

如需快速生成，可點擊**快速範本**區塊中的範本名稱：
- 月度總結
- 季度回顧
- 專案報告
- 客戶匯報

> **提示**：快速範本會自動填入建議的報告類型和日期範圍

#### Step 5：生成報告

1. 確認已選擇至少一個報告類型
2. 點擊按鈕「**📥 生成 X 份報告**」（X 為已選擇的報告數量）
3. 系統會顯示「⏳ 生成中...」狀態
4. 生成完成後，報告會出現在下方的「最近生成的報告」表格中

### 2.4 查看歷史報告

頁面下方「最近生成的報告」表格會顯示：

| 欄位 | 說明 |
|------|------|
| 報告名稱 | 報告的名稱，包含生成年月 |
| 格式 | PDF / Excel / PPTX |
| 生成時間 | 報告生成的日期時間 |
| 狀態 | 已完成 / 生成中 / 失敗 |
| 操作 | 點擊「下載」可下載報告檔案 |

### 2.5 報告生成限制

- **最少選擇**：每次至少選擇 1 個報告類型
- **日期範圍**：開始日期必須早於結束日期
- **最大生成數**：單次最多生成 6 份報告（全部類型）

## 3. 系統設定

### 3.1 頁面位置

**URL**：`/reports/settings`

### 3.2 功能總覽

系統設定分為兩大類：**一般設定**與**外觀設定**

### 3.3 一般設定

#### 自動重新整理

| 設定 | 說明 |
|------|------|
| 開啟 | 儀表板資料會自動更新 |
| 關閉 | 需要手動重新整理頁面 |

> **預設值**：開啟

#### 重新整理頻率

當「自動重新整理」開啟時可設定：

| 選項 | 頻率 |
|------|------|
| 15 秒 | 每 15 秒更新一次 |
| 30 秒 | 每 30 秒更新一次（預設） |
| 1 分鐘 | 每 1 分鐘更新一次 |
| 5 分鐘 | 每 5 分鐘更新一次 |

#### 貨幣單位

設定財務數值的顯示貨幣：

| 選項 | 說明 |
|------|------|
| 新台幣 (TWD) | 新台幣計價（預設） |
| 美元 (USD) | 美元計價 |
| 人民幣 (CNY) | 人民幣計價 |
| 歐元 (EUR) | 歐元計價 |

#### 通知提醒

| 設定 | 說明 |
|------|------|
| 開啟 | 出現異常時會發送通知 |
| 關閉 | 不發送通知 |

> **預設值**：開啟

### 3.4 外觀設定

#### 主題模式

| 模式 | 說明 |
|------|------|
| 🌙 深色 | 深色背景，白色文字（預設） |
| ☀️ 淺色 | 淺色背景，深色文字 |

> **提示**：點擊按鈕即可即時切換主題

#### 語言

設定介面顯示語言：

| 選項 | 說明 |
|------|------|
| 繁體中文 | 繁體中文介面（預設） |
| 簡體中文 | 簡體中文介面 |
| English | 英文介面 |
| 日本語 | 日文介面 |

### 3.5 儲存設定

完成設定後，點擊「**💾 儲存設定**」按鈕儲存變更。

如需恢復預設值，點擊「**↩️ 重設預設**」按鈕。

## 4. 常見問題

### Q1：為什麼生成的報告按鈕是灰底的？

這是因為您還沒有選擇任何報告類型。請先在「選擇報告類型」區塊中至少選擇一個報告。

### Q2：可以一次生成多個報告嗎？

可以！在「選擇報告類型」區塊中可以同時選擇多個報告類型，系統會一次性生成所有選擇的報告。

### Q3：生成的報告可以修改嗎？

目前版本的報告生成功能為**一鍵生成**，不支援線上編輯。如需修改報告內容，請下載後使用對應的編輯軟體（PDF Editor / Excel / PowerPoint）進行修改。

### Q4：設定的語言和主題會套用到所有頁面嗎？

會的！語言和主題設定是全系統性的，套用後所有頁面都會使用相同的設定。

### Q5：「自動重新整理」適用於哪些資料？

自動重新整理適用於所有動態資料，包括：
- KPI 數值
- 圖表數據
- 異常告警訊息
- 庫存水位

### Q6：忘記儲存設定怎麼辦？

如果沒有點擊「💾 儲存設定」而直接離開頁面，設定不會生效。下次進入時仍會看到之前的設定值。

### Q7：支援匯出 PPTX 格式嗎？

支援！PDF、Excel、PPTX 三種格式都支援。PPTX 格式特別適合用於客戶簡報或內部會議演示。
